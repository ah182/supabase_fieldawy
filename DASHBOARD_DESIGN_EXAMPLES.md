## 🎨 الألوان المستخدمة

### **Primary Colors**
- 🔵 Blue: `Colors.blue` - المنتجات
- 🟢 Green: `Colors.green` - العروض، الترندات
- 🟣 Purple: `Colors.purple` - المشاهدات، التوزيع الجغرافي
- 🔴 Red: `Colors.red` - الأخطاء

### **Neutral Colors (Light Mode)**
- ⚪ White: `Colors.white` - الخلفيات
- ⬜ Light Grey: `Colors.grey[100]` - الخلفيات الثانوية
- ⬛ Dark Grey: `Colors.grey[600]` - النصوص الثانوية
- ⚫ Black: `Colors.black87` - النصوص الرئيسية

### **Neutral Colors (Dark Mode)**
- ⚫ Dark Grey: `Colors.grey[850]` - الخلفيات
- ⬛ Medium Grey: `Colors.grey[800]` - الخلفيات الثانوية
- ⬜ Light Grey: `Colors.grey[400]` - النصوص الثانوية
- ⚪ White: `Colors.white` - النصوص الرئيسية

## 📐 المسافات والأحجام

### **Spacing**
- Mobile: `16px` padding
- Tablet: `20px` padding
- Desktop: `24px` padding

### **Border Radius**
- Cards: `16px`
- Buttons: `12px`
- Small Elements: `8px`

### **Shadows**
- Light Mode: `blurRadius: 10, offset: (0, 4)`
- Dark Mode: `blurRadius: 20, offset: (0, 10)`

### **Font Sizes**
- Headline: `20-24px`
- Title: `16-18px`
- Body: `13-15px`
- Caption: `11-12px`

## 🚀 التأثيرات البصرية

### **Gradients**
```dart
// Header Gradient (Light)
LinearGradient(
  colors: [primaryColor, primaryColor.withOpacity(0.8)]
)

// Header Gradient (Dark)
LinearGradient(
  colors: [primaryColor.withOpacity(0.8), primaryColor.withOpacity(0.6)]
)
```

### **Shadows**
```dart
// Card Shadow (Light)
BoxShadow(
  color: Colors.grey.withOpacity(0.1),
  blurRadius: 10,
  offset: Offset(0, 4),
)

// Card Shadow (Dark)
BoxShadow(
  color: Colors.black.withOpacity(0.2),
  blurRadius: 10,
  offset: Offset(0, 4),
)
```

### **Borders**
```dart
// Subtle Border
Border.all(
  color: color.withOpacity(0.2),
  width: 1,
)
```
