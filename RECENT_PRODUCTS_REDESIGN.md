## 🎨 التصميم الجديد:

### **البنية:**
```
┌─────────────────────────────────────────┐
│ اسم المنتج الكامل (سطرين)    [Badge]  │
│ السعر    👁️ المشاهدات    [🕐 الوقت]   │
└─────────────────────────────────────────┘
```

### **العناصر:**
1. **الاسم:** سطرين، حجم 14px، bold
2. **Badge النوع:** ملونة (منتج، كورس، كتاب، إلخ)
3. **السعر:** رمادي داكن، حجم 13px
4. **المشاهدات:** Badge خضراء صغيرة
5. **الوقت:** Badge رمادية خفيفة مع أيقونة ساعة
