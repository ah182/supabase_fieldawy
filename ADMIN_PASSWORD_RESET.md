# دليل إعادة تعيين كلمة المرور (للمشرف فقط)

بما أننا نستخدم نظام الإيميلات الوهمية لتوفير التكاليف، يتم تغيير كلمة المرور للمستخدمين يدوياً عبر لوحة تحكم Supabase باتباع الخطوات التالية:

## الخطوات:
1. افتح لوحة تحكم [Supabase](https://supabase.com/).
2. انتقل إلى قسم **SQL Editor** من القائمة الجانبية.
3. قم بلصق الكود أدناه مع استبدال البيانات المطلوبة.
4. اضغط على زر **Run**.

## كود SQL لتغيير كلمة المرور:

```sql
-- 1. استبدل 'NEW_PASSWORD' بكلمة المرور الجديدة التي تريد إعطاءها للمستخدم
-- 2. استبدل '010xxxxxxx' برقم هاتف المستخدم (الواتساب)

UPDATE auth.users 
SET encrypted_password = crypt('NEW_PASSWORD', gen_salt('bf')) 
WHERE email = '010xxxxxxx@fieldawy.com';
```

---

### ملاحظات هامة:
*   تأكد من كتابة رقم الهاتف بشكل صحيح كما هو مسجل في التطبيق.
*   بعد تشغيل الأمر، سيتمكن المستخدم من الدخول فوراً بكلمة المرور الجديدة.
*   يُنصح بطلب صورة الهوية من المستخدم عبر واتساب قبل تغيير كلمة المرور للتأكد من ملكيته للحساب.
